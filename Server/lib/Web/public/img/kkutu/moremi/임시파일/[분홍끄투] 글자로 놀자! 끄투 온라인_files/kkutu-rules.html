분홍끄투 5차 운영정책 (2020.03.23)

1. (목적) 이 운영정책은 분홍끄투 서버 내의 분쟁 / 싸움 / 사건 / 사고 등을 최소화하고, 건전하고 평화로운 서버를 만들기 위해 제작되었습니다.

2. (시행일) 이 운영정책은 2020.03.23 운영정책 변경 안내 이후부터 적용됩니다.

3. (용어의 정의) "서버" 라 함은 "분홍끄투 서버" 를 뜻하고, "운영자" 라 함은 관리 권한을 가지고 있는 사용자를 뜻합니다. (예: 유저관리부 / 운영자) "회원" 이라 함은 분홍끄투에 아무 로그인 수단을 이용하여 로그인을 진행한 자를 뜻하고, "총괄" 이라 함은 "분홍꽃" 을 뜻하며, "손님" 또는 "게스트" 는 서버에 로그인하지 않은 자를 뜻합니다.

4. (제재 대상)
"자" 또는 특별한 대상이 없을 경우 모든 "회원" 과 "손님" 을 뜻합니다.
① 서버 내에서 도배를 하여 다른 사용자에게 피해를 입힌 자
② 서버 내에서 타 사용자에게 욕설 / 비속어 / 비하 발언을 사용한 자 (혼자 있을 경우에는 욕설은 가능)
③ 서버 내에서 운영자를 사칭하여 타 사용자에게 혼란을 준 자
④ 서버 내에서 운영자에게 무리한 요구 등을 한 자
⑤ 서버를 공격한 자 (DoS / DDoS 등)
⑥ "핵" 모드 (특수규칙) 를(을) 사용하지 않고 핵 / 매크로 / 자동입력 프로그램 등을 사용한 회원
⑦ 닉네임에 부적절한 문구가 들어가 있는 회원
⑧ 분쟁 / 싸움 등을 유도한 자
⑨ 버그 악용 / 비정상적인 경로 등으로 경험치 / 핑을 획득한 회원
⑩ 게임 내에서 사기 행위를 한 자
⑪ 타 서버를 홍보하는 발언을 한 자 (언급만으로는 해당되지 않습니다.)
⑫ 서버 내에서 매크로를 사용하여 지속적인 방 제작, 아이템 구매 등을 하여 서버를 다운시킨 회원 (큰 피해가 없을 경우 제외)
⑬ 관리/단어부 해당 본인의 권한을 남용하여 타 사용자 또는 본인의 계정을 임의로 조작한 회원 또는 사전에 전혀 존재하지 않는 단어 및 추가를 자제하라고 공지했던 단어를 지속적으로 추가하는 회원
⑭ 총괄 재량


5. (제재 내용)
①, ②... 는 4. (제재 대상) 의 ①, ②... 를 뜻합니다.
제재 대상이 손님일 경우 IP정지 (기간: 7일~무기한)
2차 이상일 경우 가장 높은 경고 횟수대로 제재 또는 운영자 재량으로 제재하며, 횟수는 월말에 초기화됩니다.
기한이 쓰여 있지 않을 경우 무기한을 뜻합니다.

① 1차: 채팅정지 3~7일, 2차: 채팅정지 7~21일, 3차: 계정 무기한 정지
② 1차: 정도에 따라 운영자 재량으로 최소 채팅정지 7일~최대 계정 무기한 정지
③ 1차: 계정 무기한 정지, 2차: IP 정지
④ 1차: 채팅정지 7~14일, 2차: IP 정지
⑤ 1차: IP 무기한 차단

⑦ 1차: 닉네임 초기화, 2차: 1차와 동일 + 게임이용 14~21일 정지, 3차: 1차, 2차와 동일 + 계정 무기한 정지
⑧ 1차: 채팅정지 7~14일, 2차: 계정 무기한 정지
⑨ 1차: 계정의 경험치 / 핑 80% 초기화, 2차: 계정의 경험치 / 핑 전체 초기화, 3차: 계정의 경험치 / 핑 / 보유 & 착용 아이템 초기화, 4차: 계정정지 7~14일, 5차: 계정 무기한 정지
⑩ 1차: 계정 무기한 정지, 2차: IP 정지

⑫ 1차: 아이템 구매 요청일 경우 해당 아이템 1개 제외 전체 회수 + 게임이용 7~21일 제한 (공통) / 1차와 동일 + 계정 무기한 정지
⑬ 계정 무기한 정지 + 권한 박탈 / 조작한 계정이 운영자의 계정일 경우 계정 무기한 정지 + IP 무기한 차단 + 권한 박탈
⑭ 총괄 재량으로 제재


6. (변경 시 안내)
운영정책은 안내 후 시행하며, 시행 전에 발생한 문제는 시행 전 운영정책대로 제재합니다.

7. (제재 시 안내)
회원이 운영정책을 위반하여 제재될 경우, 운영자는 회원에게 제재 사실을 알리고 해당 회원을 서버에서 내보냅니다. 디스코드 서버의 제재-내역에 제재 내용과 사유가 올라가며, 디스코드에 접속이 불가능한 회원은 카페의 질문게시판에서 제재 사유, 기간 등을 요청하여 확인 가능합니다.

8. (제재 예외 및 제재기간 도중 해제)
• 총괄은 공지 후 제재 기간 중의 회원의 제재를 해제할 수 있습니다.
• 2개 이상의 사항이 겹치는 경우, 운영자 재량으로 제재의 강도를 추가로 높일 수 있습니다.
• 총괄은 서버 관리, 운영 등을 위해 제재 대상에서 제외됩니다. 하지만 최대한 해당 사안을 어기지 않도록 노력해야 합니다.

9. 관리 / 단어부 해당 (관리 / 단어부 추가 제재)
위에 명시된 사항 이외에도 아래에 추가로 명시한 사항을 지키지 않을 경우 제재를 받을 수 있습니다.
아래 사항은 총괄의 허가를 받았을 경우에는 제외됩니다
9-1. 허락 없이 정상적인 단어를 삭제하는 행위 (단, 윰차, 릇데껌 등의 밸런스 붕괴가 생길 수 있는 단어는 제외)
9-2. 타 부서의 비밀번호를 사용하여 본인의 권한 밖의 행동을 하는 행위
9-3. 특별한 사유 없이 회원을 제재하는 행위
9-4. 하지 말라고 경고했던 행위를 하여 서버를 다운시키는 행위
9-5. 총괄 재량

해당 행위가 적발될 경우 경고 없이 모두 총괄 재량으로 제재합니다.

10. (제재 강도 조정)
총괄은 제재 대상자가 큰 피해, 또는 작은 피해를 주었을 때, 본인 재량으로 제재 강도를 조정할 수 있습니다.
